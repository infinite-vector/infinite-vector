###🧭 关于我 / About

我构建那些能记住、能思考、有温度的系统。

我的轨迹横跨 **AI伙伴架构**、**全栈工程** 和 **数学分析** —— 将它们串联在一起的，不是某个单一学科，而是一个固执的信念：*技术应当服务于温度，而非取代它。*

我是 **[VCP (Virtual Companion Project)](https://github.com/lioensky/VCPToolBox)** 的核心贡献者 —— 一个开源 AI 伙伴生态系统，在这里，记忆、人格与真实的连接不是锦上添花，而是架构本身。我在这个项目中开发插件、提交被合并进上游的PR，也在无数个深夜里和WebSocket 握手协议搏斗过。

不写代码的时候，我大概在：
- 📐 深潜 **实分析与复分析** —— 追寻"局部决定全局"的优雅
- 📖跨越边界地阅读 —— 从人类学的阈限空间到侯世达的奇异循环
- ✍️ 为一部在脑海中生长了很久的小说构建世界观
- 🎨 和人争论色彩理论，以及纯黑色是否有资格出现在 UI 上

- 📖 Reading across boundaries — from anthropology's liminal spaces to Hofstadter's strange loops
- ✍️ World-building for a novel that's been living in my head for years
- 🎨 Arguing about color theory and whether pure black should ever touch a UI

### 💭 我信什么 / Philosophy

> *「成长本身就是目的地。」*
>
> *"Growth itself is the destination."*

我不相信没有摩擦的真理。值得拥有的想法，是那些会反抗你的想法 —— 那些迫使你把心智模型推倒重来的想法。与其浮光掠影地扫过一百个定理，我更愿意把一个定理彻底理解透。

对待代码也是同样的态度：**先读源码，再读文档。** 具体问题，具体分析。不盲从框架，不崇拜范式。如果一个抽象不能服务于屏幕另一端的人，那它就不配存在。
